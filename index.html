2-4 Subagents — 독립 AI 에이전트 위임

🤖
한 줄 정의
Subagent = "독립 컨텍스트에서 전문 작업을 수행하는 하위 AI"
메인 에이전트가 지휘하고, 서브에이전트가 전문 실행합니다.
특징설명
독립 컨텍스트메인 대화 오염 없이 대용량 작업 처리
병렬 실행여러 에이전트 동시 실행 → 속도 향상
최소 권한역할별 필요한 도구만 부여 → 안전성
격리 모드isolation: "worktree" → 안전한 실험
💪 핸즈온 실습 — 20분
서브에이전트 정의 파일 작성
.claude/agents/code-reviewer.md 작성 → 코드 리뷰 전문 에이전트 설계 → 실제 파일로 테스트
Agent 도구 호출 예시 
// 단일 에이전트
Agent({
  description: "코드 리뷰",
  prompt: "auth.js 보안 검토",
  subagent_type: "code-reviewer"
})

// 병렬 에이전트 (동시 실행)
Agent({ prompt: "프론트엔드 분석" })
Agent({ prompt: "백엔드 분석" })
Agent({ prompt: "DB 스키마 분석" })
// → 3개 동시 실행, 결과 취합
⚡
병렬 처리
독립 작업은 동시 실행 → 3배 빠른 처리
🔒
격리 실행
worktree 모드로 메인 코드 보호

2-5 MCP — 외부 서비스 연결 표준 프로토콜

claude mcp add slack --scope user
claude mcp add github --scope project
claude mcp add discord --scope project
🔌
한 줄 정의
MCP = "Claude Code가 외부 서비스와 통신하는 표준 프로토콜"
3,000+ 서버 생태계 — 이미 만들어진 연동을 바로 사용.
서비스실전 활용
💬 Discord / Slack작업 완료 → 채널 자동 보고
🐙 GitHub이슈 자동 관리 · PR 리뷰
📁 Google Drive보고서 → 팀 드라이브 자동 저장
📧 Gmail완성 문서 → 고객사 자동 발송
🎨 Canva콘텐츠 → 발표자료 자동 생성
scope 3단계
user — 내 모든 프로젝트에서 사용
project — 이 프로젝트 팀 전체 공유
local — 나만, 이 프로젝트에서만
